Newton Robot Capstone Project

Open-source quadruped robot with ML capabilities

Project Overview

Newton is an open-source quadruped robot developed as a capstone project with a focus on accessibility and innovation. The robot features advanced machine learning capabilities for walking algorithms and a custom-developed BLDC motor system with integrated drivers and encoders. The primary objective is to lower the barrier of entry for students, researchers, and hobbyists interested in robotics and legged locomotion.

Built on the foundation of modular design principles, Newton provides a comprehensive platform for experimentation in autonomous movement, sensor integration, and motor control. Every component has been carefully designed to be reproducible and accessible.

Hardware Development

Custom BLDC motor design optimized for quadruped locomotion. Integrated motor drivers and high-precision encoder systems enable accurate position feedback and coordinated leg movement. The frame is constructed using lightweight materials to maximize efficiency.

  • BLDC Motors
  • Custom Motor Drivers
  • Rotary Encoders
  • Microcontroller Integration

Software & ML

Newton uses Python-based machine learning algorithms to optimize walking patterns and adapt to different terrain conditions. The system learns efficient gait parameters through reinforcement learning and simulation.

  • Python ML Frameworks
  • Gait Optimization
  • Real-time Control Loop
  • Sensor Data Processing

Communication Protocols

Multi-protocol support allows seamless integration with various microcontrollers and sensors. The system uses I2C for sensor communication and CANbus for motor control synchronization across all four legs.

  • I2C Protocol
  • CANbus
  • Serial Communication
  • Sensor Fusion
